## Setup & Run
```bash
git clone https://github.com/tejasredefine/playwright-registration.git
cd playwright-registration
npm install
npm run install-browsers
npm start
```

## Claude Desktop Config
Place `claude_desktop_config.json` at:
```
C:\Users\<YourUsername>\AppData\Roaming\Claude\claude_desktop_config.json
```
> Replace `<YOUR_MYSQL_PASSWORD>` and `<YOUR_PG_PASSWORD>` with actual passwords.
